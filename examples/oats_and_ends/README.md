# oats and ends example

Serves the "Oats & Ends" page, built from the shared
[fixtures](../../tests/fixtures/) crate, using
[axum](https://docs.rs/axum/latest/axum/),
[tokio](https://docs.rs/tokio/latest/tokio/)

To run the example, use the command `cargo run -p oats_and_ends`. Once running,
open `http://127.0.0.1:8080/` in your browser.
